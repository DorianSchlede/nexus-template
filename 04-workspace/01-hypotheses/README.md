# Hypotheses

Hypothesis tracking organized by category.

## Categories

- `desirability/` - Customer pain & demand hypotheses (D##)
- `feasibility/` - Technical feasibility hypotheses (F##)
- `viability/` - Business model hypotheses (V##)

## Naming Convention

`X##-short-name.md` (e.g., `D1-root-cause-blindness.md`)

## Status Tracking

Each hypothesis file tracks:
- Statement
- Evidence for/against
- Validation status
- Next actions

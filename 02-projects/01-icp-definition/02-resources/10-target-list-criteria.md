## Filterkriterien Template

### MUST HAVE (Dealbreaker if missing)

```
□ Industry: [Deine Ziel-Industries]
  Examples: SaaS, FinTech, E-Commerce, MarTech

□ Company Size: [Dein Sweet Spot]
  Examples: 50-500 employees, $5M-$50M ARR

□ Geography: [Deine Märkte]
  Examples: DACH, US, UK, Global

□ AI/Automation: [Minimum tech maturity]
  Examples: Already using AI, automation tools in place

□ Funding/Revenue: [Economic viability]
  Examples: Series A+, $1M+ ARR, profitable
```

### NICE TO HAVE (Bonus points)

```
□ Recent Funding: Series A/B/C in last 12 months
□ Growth Mode: Hiring aggressively, expanding
□ Known Pain Points: Public complaints about current solutions
□ Competitor Usage: Using similar tools (replacement opportunity)
□ Tech Stack Fit: Already using complementary tools
□ Executive Change: New CTO/VP Ops in last 6 months
```

### EXCLUDE (Anti-ICP)

```
□ Too Small: <20 employees, pre-revenue
□ Too Large: >5000 employees (unless specific segment)
□ Wrong Industry: [Your exclusion list]
□ Wrong Geography: Regions you can't serve
□ Known Bad Fit: Competitors' employees, agencies
```

## Tiered Prioritization

### Tier 1: Perfect Fit (5-10 companies)
```
Criteria:
  - Matches ALL must-have criteria
  - Has 3+ nice-to-have signals
  - Active buying signal present
  - Known contact available

Action:
  - Personalized outreach
  - Multi-threaded approach
  - Executive sponsorship
  - Custom content/demo
```

### Tier 2: Good Fit (10-20 companies)
```
Criteria:
  - Matches ALL must-have criteria
  - Has 1-2 nice-to-have signals
  - No active buying signal

Action:
  - Personalized outreach
  - Champion-focused
  - Standard demo flow
  - Nurture if no response
```

### Tier 3: Potential Fit (20-50 companies)
```
Criteria:
  - Matches most must-have criteria
  - May have questions about fit

Action:
  - Semi-personalized outreach
  - Qualify quickly
  - Move to Tier 2 or disqualify
```

## List Building Process

### Step 1: Define Criteria (30 min)
- Lock down must-have filters
- Define nice-to-have bonus criteria
- Set exclusion rules

### Step 2: Initial Search (2-4 hours)
- Use LinkedIn Sales Nav / Apollo / Crunchbase
- Apply must-have filters
- Export initial list (100-200 companies)

### Step 3: Enrichment (2-4 hours)
- Research tech stack (BuiltWith/Wappalyzer)
- Check for buying signals (G2, job postings)
- Verify company details

### Step 4: Scoring & Tiering (1-2 hours)
- Apply ICP scoring rubric
- Assign tiers based on score
- Rank within tiers

### Step 5: Contact Research (4-8 hours)
- Identify champion candidates
- Find economic buyers
- Verify contact information
- Note LinkedIn connections

### Step 6: Outreach Planning
- Tier 1: Personalized sequences
- Tier 2: Semi-personalized templates
- Tier 3: Batch qualification

---

## Maintenance

### Weekly
- Add new companies from inbound/referrals
- Update outreach status
- Move companies between tiers based on engagement

### Monthly
- Refresh buying signals
- Remove stale prospects
- Add new companies from research

### Quarterly
- Full list audit
- Update ICP criteria if needed
- Analyze conversion by tier

---

## Quick Qualification Questions

When researching a company, answer:

1. ✅ Do they match our must-have criteria?
2. ✅ Can we identify a potential champion?
3. ✅ Is there a reason they'd buy NOW?
4. ✅ Do we have a path to reach them?
5. ❌ Are there any Anti-ICP red flags?

**If 4+ checks = Add to list**
**If 2-3 checks = Tier 3 for quick qualification**
**If <2 checks = Skip**

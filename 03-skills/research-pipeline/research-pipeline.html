üèóÔ∏è 7-Level Hierarchical Architecture

6 out of 7 levels are Python scripts (not LLMs) - only L4 and L7 use AI

Level 1: Routing (Script)
Boolean lookup using chunk_index - which chunks have which fields? Creates routing table to avoid sending irrelevant chunks to subagents (saves 60% tokens)
Level 2: Allocation (Script)
Greedy bin-packing algorithm batches chunks into 70k-token groups while reserving 30k for L7 report generation
Level 3: Prompt Generation (Script)
Generates INPUT CONTRACT per batch containing briefing + extraction guide + specific chunk list
Level 4: Pattern Extraction (15 Parallel Subagents) ‚ö° LLM
Each subagent reads its prompt file and extracts cross-paper patterns with chunk:line citations
Level 5: Verification (Script)
Spot-checks 10% of quotes to verify they exist at cited chunk:line locations (¬±5 line tolerance)
Level 6: Aggregation (Script)
Fuzzy deduplication (90% similarity threshold) merges duplicate patterns across batches and ranks by evidence count
Level 7: Final Report (Subagent) ‚ö° LLM
Narrative synthesis identifying cross-field insights and writing human-readable markdown report
